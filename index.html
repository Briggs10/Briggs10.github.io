Mustang Bucks - Interactive browser game built with HTML, CSS and JavaScript

🏁 Mustang Bucks – Hollywood Beach Edition an interactive browser-based driving game where players enter three numbers between 100–999 for a chance to win a 2027 Ford Mustang or $200 Ford Bucks'.

🎮 Game Overview: Mustang Bucks is a canvas-powered arcade-style web game built using HTML, CSS, and JavaScript. Players enter a driver’s name, select a vehicle, input three 3-digit numbers. Attempt to match the dealer’s numbers across three rounds. Win all three rounds to drive away with a Mustang. Crashed Out? You still have a chance to win $200 Ford Bucks in Round 3.

🚗 Features: Animated highway and beach environment, Day-to-night scene transition, Moving vehicle progression system, Round-based win tracking, Confetti celebration effect, Dynamic dealership ticket, Generation responsive canvas layout, and Local expiration date logic (90-day ticket validity).

🛠️ Built With: HTML5 Canvas, CSS3, and JavaScript. Using client-side game state logic, Randomized number generation algorithm. No frameworks. No libraries. Pure front-end implementation.

🔮 Future Enhancements: Sound effects (engine rev, crash, celebration), Difficulty modes, Score leaderboard, Persistent high-score tracking, Expanded vehicle selection, Mobile optimization improvements.
